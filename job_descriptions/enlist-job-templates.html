Job description templates for a wide variety of technical roles. Each description has a basic description of the role, a bulleted list of responsibilities, and a list of requirements for the position. These are good places to start if you're not even sure where to begin. Templates are available for the following roles:

Data Scientist Embedded Systems Engineer React Developer Mobile Developer DevOps Engineer Network Administrator Back-end Engineer Cybersecurity Specialist Systems Engineer Data Analyst Database Administrator Game Developer Front-end Engineer

For those who are looking for them, Enlist also offers templates for finance, marketing, legal, operations, H.R., design, and healthcare.
